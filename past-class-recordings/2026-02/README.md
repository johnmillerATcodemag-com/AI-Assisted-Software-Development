# February 2026 Class Recordings

VTT transcript files for classes held in February 2026.

## Files

Place VTT files in this directory with the naming format:
- `[Course Name] ([Day] [Time Period]).vtt`

## Analysis

To analyze a VTT file in this directory:

```
/run summarize-vti-content vti_file="past-class-recordings/2026-02/[filename].vtt"
```

The summary will be generated as `[filename]-summary.md` in this same directory.
